== Description ==con solo colocar este widget, tendrás actualizado tu sitio web todos lo dias, ya que bibliatodo se encarga de colocar el la mejor imagen del día, edificante para tus visitantes.== Installation ==Puedes colocar la Imagen del día sin necesidad de instalar este plugin, con solo copiar este codigo en tu widget como Texto arbitrario o HTML:`<script type="text/javascript" language="javascript" src="https://cdn.bibliatodo.com/assets/js/wordpress/es/widget-imagen-dia.js"></script>`o puedes instalar el el plugin y seguir estas indicaciones
1. Sube el archivo a wp-content/plugin2. Activar plugin3. Ir a Apariencia > widgets4. Arrastra el widget al sidebar que quieras.5. Comprueba que se ve correctamente!
   - Widget: Se puede añadir un widget a la barra lateral para que se muestre la Imagen del día, vaya a apariencia - widget "Imagen del día"
   - Template: en cualquier lugar de su plantilla inserte este codigo `<script type="text/javascript" language="javascript" src="https://www.bibliatodo.com/assets/js/wordpress/es/widget-imagen-dia.js"></script>`.
